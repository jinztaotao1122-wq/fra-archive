STEP 01
Create Google Sheet for Metadata
Go to sheets.google.com → New Spreadsheet → Name: FRA Archive Data. Add these Row 1 headers:
id year month type center school name lesson level activity comments fileIds fileUrls timestamp
STEP 02
Create Google Drive Folder
Create a folder FRA Photos in Google Drive. Set sharing to "Anyone with the link → Viewer". Copy the folder ID from the URL after /folders/.
STEP 03
Deploy Apps Script
In the Sheet → Extensions → Apps Script → paste this code → Deploy → New Deployment → Web App → Execute as: Me, Access: Anyone → Copy URL.
